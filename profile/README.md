## Projects

### nnsight

<p align="left">
  <img src="https://github.com/ndif-team/nnsight/blob/main/nnsight_logo.svg" alt="nnsight" width="120">
</p>

A Python library for interpreting and manipulating the internals of deep learning models. Access activations, modify them to study causal effects, compute gradients, and batch interventions efficiently.

[Documentation](https://www.nnsight.net) | [GitHub](https://github.com/ndif-team/nnsight) | [Paper](https://arxiv.org/abs/2407.14561)

```bash
pip install nnsight
```

### nnterp

<p align="left">
  <img src="https://github.com/ndif-team/nnterp/blob/main/nnterp_logo.png" alt="nnterp" width="120">
</p>

A unified interface for mechanistic interpretability of transformers. Built on NNsight, it provides standardized naming conventions across all transformer architectures with built-in interventions like logit lens, patchscope, and activation steering.

[Documentation](https://ndif-team.github.io/nnterp/) | [GitHub](https://github.com/ndif-team/nnterp) | [Paper](https://arxiv.org/abs/2511.14465)

```bash
pip install nnterp
```
